# CRM con Clientes y Notas

Una aplicación web sencilla para la gestión de clientes y sus notas correspondientes, desarrollada con tecnologías web fundamentales.

## 🌐 Demo en Vivo

Puedes probar la aplicación aquí: [CRM App - Demo](https://app-lu561-nid68wien-luisernestosalas-2775s-projects.vercel.app)

## 📋 Características

- **Gestión de Clientes**: Crear, editar, visualizar y eliminar registros de clientes
- **Sistema de Notas**: Asociar múltiples notas a cada cliente
- **Interfaz Intuitiva**: Diseño limpio y fácil de usar
- **Almacenamiento Local**: Los datos se guardan en el navegador del usuario
- **Responsive Design**: Funciona en dispositivos móviles y escritorio

## 🛠️ Tecnologías Utilizadas

- **HTML5**: Estructura y semántica de la aplicación
- **CSS3**: Estilos y diseño responsivo
- **JavaScript (Vanilla)**: Lógica de negocio y manipulación del DOM
