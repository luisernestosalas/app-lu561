# Manual de Usuario - CRM con Clientes y Notas

## ¿Qué es esta aplicación?

Es una aplicación web simple para gestionar clientes y sus notas. Te permite agregar, ver, editar y eliminar información de clientes de forma fácil.

## Cómo acceder

Abre tu navegador web e ingresa a:
```
https://app-lu561-nid68wien-luisernestosalas-2775s-projects.vercel.app
```

## Funciones principales

### 1. Ver lista de clientes
- Al abrir la app verás todos los clientes registrados
- Cada cliente muestra su nombre y información básica

### 2. Agregar nuevo cliente
- Busca el botón "Agregar Cliente" o "Nuevo Cliente"
- Completa los campos requeridos:
  - Nombre
  - Email
  - Teléfono
  - Otros datos solicitados
- Haz clic en "Guardar"

### 3. Ver detalles del cliente
- Haz clic sobre el nombre de cualquier cliente
- Se mostrará toda su información
- Podrás ver las notas asociadas

### 4. Agregar notas a un cliente
- Dentro del perfil del cliente, busca "Agregar Nota"
- Escribe el contenido de la nota
- Guarda la nota
