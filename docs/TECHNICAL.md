# Documentación Técnica - CRM con Clientes y Notas

## Información General

**Aplicación:** Sistema CRM para gestión de clientes y notas  
**Tecnologías:** HTML + JavaScript (Vanilla)  
**Repositorio:** https://github.com/luisernestosalas/app-lu561  
**URL de Producción:** https://app-lu561-nid68wien-luisernestosalas-2775s-projects.vercel.app  
**Plataforma de Deploy:** Vercel  

## Stack Tecnológico

### Frontend
- **HTML5:** Estructura y semántica de la aplicación
- **JavaScript (Vanilla):** Lógica de negocio y manipulación del DOM
- **CSS:** Estilos y presentación visual

### Características del Stack
- **Sin frameworks:** Implementación con tecnologías web nativas
- **Cliente-side:** Aplicación que se ejecuta completamente en el navegador
- **Responsive:** Compatible con dispositivos móviles y desktop

## Funcionalidades Principales

### Gestión de Clientes
- Crear nuevos registros de clientes
- Visualizar lista de clientes
- Editar información de clientes existentes
- Eliminar registros de clientes

### Sistema de Notas
- Asociar notas a clientes específicos
- Crear, editar y eliminar notas
- Visualización organizada de notas por cliente
